# Happy Birthday Naawal

A small Next.js birthday website for Naawal Farah.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm install
npm run build
npm start
```

## Music

Add your licensed MP3 here:

`public/music/happy-birthday-simi-adekunle.mp3`

The site has a floating play/pause button already wired up. Browsers sometimes block autoplay until the visitor interacts with the page, so the button is the reliable fallback.

## Hosting on Hostinger

If you are using Hostinger for Node hosting:

1. Upload the project.
2. Run `npm install`.
3. Run `npm run build`.
4. Start with `npm start`.

If you are exporting or using a static setup later, this design can also be converted.
